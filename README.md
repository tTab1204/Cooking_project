# :curry: Cooking (MERN stack)
### 요리 공유 플랫폼!
사용자들이 호스트의 요리 이벤트를 예매하고, 요리사들이 호스트로 등록되는 MERN stack 웹사이트 (ES6)

## Features
- 로그인/회원가입
- JWT Authorization
- 이벤트 검색 (이름, 날짜)
- 댓글(답글), 팔로우, 좋아요/싫어요
- MyTicketPage에 상품 추가/제거
- Ant Design
- Styled Components
- 반응형 웹 페이지

## Tools
- React
- Redux
- Recoil(시범 적용)
- Node
- Express
- Mongoose
- Heroku

## Design Pattern
- Presentational / Container 컴포넌트 방식 적용

## Installation
```
$ npm install
```

## Running
```javascript
 "scripts": {
    "start": "node server/index.js",
    "dev": "concurrently \"npm run backend\" \"npm run start --prefix client\"",
  }
```
