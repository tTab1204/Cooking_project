# :curry: Cooking Project (이름 미정)
### 이 프로젝트는 미완성입니다!
요리 이벤트를 예매하고, 요리사들이 호스트로 등록되는 MERN stack 웹사이트

## Features
- 로그인/회원가입
- JWT Authorization
- 이벤트 검색 (이름, 날짜)
- 댓글(답글), 팔로우, 좋아요/싫어요
- MyTicketPage에 상품 추가/제거
- Ant Design
- Styled Components
- 반응형 웹 페이지

## Tools
- React
- Redux
- Recoil(시범 적용)
- Node
- Express
- Mongoose
- AWS (적용 예정)
